Overview

This project implements a mean reversion investment strategy using Python and Jupyter Notebook. It includes annotated code, clear explanations of data retrieval, strategy implementation, performance evaluation, and visualizations. The workflow is designed to be reproducible and transparent for both peers and potential employers.

Responsibilities

  • Structured the project into annotated Jupyter Notebook sections: data retrieval, strategy, evaluation, visuals.
  • Implemented mean reversion backtesting logic with clear documentation and reproducible workflows.
  • Created performance metrics and charts to evaluate strategy effectiveness.
  • Published the notebook and supporting files on GitHub with a detailed README.

Methods

  • Python (pandas, NumPy, Matplotlib, Plotly) for data analysis and visualization.
  • Jupyter Notebook for annotated, step-by-step documentation of the workflow.
  • GitHub repository hosting with README and reproducible instructions.
  • Mean reversion strategy backtesting with performance evaluation metrics.

Results & Impact

The backtesting results demonstrated that the mean reversion strategy significantly outperformed the DIA ETF benchmark over the 2016–2025 period. Starting from a $100,000 portfolio, the strategy grew to over $500,000, while the DIA ETF reached approximately $300,000. Despite higher volatility, the strategy delivered superior cumulative returns and showcased the potential of systematic reversion-based approaches.

The project also produced a reproducible workflow, complete with annotated code, performance metrics, and visualizations, enabling transparent evaluation and future enhancements. It provided a clear roadmap for showcasing financial modeling and data science work, and created a professional GitHub repository for sharing with peers, employers, and online communities.
