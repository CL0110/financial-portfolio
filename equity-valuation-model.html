Overview

Developed a comprehensive driver-based DCF framework tailored to tech companies. Built revenue, margin, capex, and working capital schedules with clear audit trails and input sheets, enabling transparent assumption changes.

Responsibilities

  • Constructed integrated three-statement model with free cash flow outputs.
  • Implemented scenario and sensitivity analyses on growth, margins, WACC, and terminal assumptions.
  • Triangulated results using trading comps and precedent transactions to validate the valuation range.
  • Delivered investor-ready Excel model with documentation and versioned assumptions.

Methods & Stack

  • DCF with explicit forecast horizon and terminal value via Gordon Growth and exit multiple cross-checks.
  • Scenario manager for base/bull/bear drivers; data validation and input gating.
  • Excel/VBA for automation, sensitivity tables, and charting of valuation bridges.

Results & Impact

Produced a defensible valuation range with quantified sensitivities. Accelerated turnaround time for updates, improved comparability across coverage, and standardized communication of key drivers to stakeholders.
